# @hyas/cli
[![npm](https://img.shields.io/npm/v/@hyas/cli.svg?style=flat-square)](https://www.npmjs.com/package/@hyas/cli)

Hyas command line tool.

## Installation

Install Hyas CLI globally:

```bash
npm install -g @hyas/cli
```

## Commands

Create new website:

```bash
hyas create [dir]
```

Start development server:

```bash
hyas start
```

Oher commands:

```bash
hyas help
```
